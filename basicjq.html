Q.1 What is jQuery?

Ans:- jQuery is a popular, fast, and concise JavaScript library designed to simplify HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. It allows developers to write less code while achieving more in terms of interactivity and functionality on their web pages.

One of the main advantages of jQuery is its ability to achieve cross-browser compatibility, meaning that the same code will work consistently across different web browsers. This simplifies development and testing, as developers don't need to write separate code for different browsers.

jQuery simplifies the process of manipulating HTML documents and handling events, making it easier to create dynamic and interactive web applications. It provides a wide range of features and utilities, such as DOM manipulation, CSS manipulation, event handling, animation effects, and AJAX functionality, all accessible via a simple and intuitive API.

Overall, jQuery has been widely adopted by web developers due to its ease of use, versatility, and extensive community support, making it a powerful tool for building modern web applications.

Q.2:-How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

Ans:- Certainly! Here's how you can perform CSS manipulation, add and remove classes, and create animations using jQuery:

Applying CSS using jQuery:

You can use the `.css()` method to apply CSS styles to HTML elements.

// Syntax: $('selector').css('property', 'value');
// Example: Change the color of all paragraphs to red
$('p').css('color', 'red');

Adding and removing classes using jQuery:

jQuery provides methods like `addClass()` and `removeClass()` to add or remove classes from HTML elements.

// Syntax to add a class: $('selector').addClass('classname'); // Syntax to remove a class: $('selector').removeClass('classname');
// Example: Add a class 'highlight' to all paragraphs
$('p').addClass('highlight');
// Example: Remove a class 'active' from all divs
$('div').removeClass('active');

jQuery Animation:

jQuery allows you to create animations on HTML elements using methods like `animate()`. You can animate properties like width, height, opacity, etc.

// Syntax: $('selector').animate({properties}, duration);
// Example: Animate the width of a div to 300px over 1 second
$('div').animate({width: '300px'}, 1000);

These are basic examples to get you started. jQuery offers more advanced features for CSS manipulation and animation, allowing you to create dynamic and visually appealing web pages with ease.
